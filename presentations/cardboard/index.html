Google Cardboard

Created by Seth Clark / +Seth

Tools to use:

  • Android SDK
  • Unity 3D
  • Three.js
  • LibGDX

Android SDK provides:

  • Lens distortion correction.
  • Head tracking.
  • 3D calibration.
  • Side-by-side rendering.
  • Stereo geometry configuration.
  • User input event handling.

Unity Plugins provide:

  • User head tracking
  • Side-by-side stereo rendering
  • Automatic stereo configuration for a specific Cardboard model
  • Distortion correction for Cardboard lenses
  • Support for Image Effects and Deferred Rendering, or bypassing this support for increased FPS.
  • Interaction with Canvas UI elements using gaze and the trigger.
  • Simulate a user's head movement in Unity play mode using the mouse and alt/control keys to pan or tilt the camera.
  • ...

Using the Android SDK

First, you'll need to add the dependencies to your project' lib folder

  • cardboard.jar
  • libprotobuf-java-2.6-nano.jar

Both are found in the Cardboard Sample Github page.

Add to your build.gradle.


dependencies {
     ...
     compile fileTree(dir: 'libs', include: ['*.jar'])
}

Modify your manifest

Add required permissions


<uses-permission android:name="android.permission.NFC" />
<uses-permission android:name="android.permission.VIBRATE" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
<uses-permission android:name="android.permission.WRITE_EXTERNAL_STORAGE" />

Declare OpenGL ES 2.0 required


<uses-feature android:glEsVersion="0x00020000" android:required="true" />

Set Min API


<uses-sdk android:minSdkVersion="16" android:targetSdkVersion="23" />

Modify your manifest

Modify your Cardboard Activity


<activity
     android:name=".MyCardboardActivity"
     android:screenOrientation="landscape">
     ...

     <intent-filter>
          <category android:name="com.google.intent.category.CARDBOARD" />
     </intent-filter>
</activity>

Define a CardboardView


<com.google.vrtoolkit.cardboard.CardboardView
     android:id="@+id/cardboard_view"
     android:layout_width="fill_parent"
     android:layout_height="fill_parent" />

CardboardView is a convenience extension of GLSurfaceView.

Create Cardboard Activity

Extend CardboardActivity


public class MyCardboardActivity extends CardboardActivity 
        implements CardboardView.StereoRenderer {
    ...
    @Override
    public void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.cardboard_layout);
        CardboardView cardboardView = 
                (CardboardView) findViewById(R.id.cardboard_view);
        
        cardboardView.setRenderer(this);
        
        setCardboardView(cardboardView);
    }
}

Render something!


public class MyCardboardActivity extends CardboardActivity 
        implements CardboardView.StereoRenderer {

    @Override
    public void onNewFrame(HeadTransform headTransform) {
        // Get head tranformation matrix
        headTransform.getHeadView(headView, 0);
    }

    @Override
    public void onDrawEye(Eye eye) {
        // Draw frame for each eye.  
        // Called twice per frame.
    }
}

Finally...Handle user input


public class MyCardboardActivity extends CardboardActivity 
        implements CardboardView.StereoRenderer {

    @Override
    public void onCardboardTrigger() {
        // Do something.
    }
}
